# video-game-sales-analysis
Анализ продаж видеоигр — SQL, Python, Tableau
# О проекте
Анализ датасета с 16 000 видеоигр — продажи по жанрам, 
платформам и регионам за 1980–2016 годы.
# Инструменты
- Python (pandas, matplotlib)
- SQL (SQLite)
- Tableau Public
# Что сделал
- Очистил данные (удалил пропуски, исправил типы)
- Написал SQL запросы для анализа
- Построил 3 графика в Python
- Сделал дашборд в Tableau
